# wikipedia-article-selector-
A simple tool to detect and display Wikipedia article titles using JavaScript
1Wikipedia Article Selector

📌 Project Overview

This project is a simple browser-based tool that detects and displays Wikipedia article titles using JavaScript.

🚀 Features

- Extracts article titles from Wikipedia pages
- Simple and clean interface
- Lightweight and fast

🛠️ Tech Stack

- HTML
- CSS
- JavaScript

📂 How to Use

1. Open the project in browser
2. Run the script
3. View extracted article titles

🌟 Future Improvements

- Chrome Extension version
- Integration with MediaWiki API
- Add UI enhancements
- Clickable article titles that open directly on Wikipedia
- - Integrated with MediaWiki API for real-time search
